### Invoice Auto-Generation

Automated system to generate monthly invoices to Centara Grand as HTML → PDF.

**Usage:**
```bash
cd payments && source venv/bin/activate
python3 scripts/generate_invoice.py 2026-02                    # auto-detect from schedule
python3 scripts/generate_invoice.py 2026-02 --invoice-no 3176  # explicit number
python3 scripts/generate_invoice.py 2026-02 --dry-run          # preview calculations
python3 scripts/generate_invoice.py 2026-01 --no-dj "CRU:1" --extra "CRU:17:17:00-19:00"
```

**How it works:**
- Reads `schedules/YYYY-MM/schedule.md` to auto-detect `**Guest DJ**` entries as deductions
- Builds line items: base charge (days × 2 venues × ฿3,711.34/evening) + deductions + partial hours
- Calculates: Sub Total → +VAT 7% → Grand Total → -WHT 3% → Net Amount
- Fills HTML template → Chrome headless PDF
- Auto-increments invoice number via `invoice_config.json`

**Chrome PDF notes:**
- Uses `@page { margin: 0 }` + wrapper div padding to suppress Chrome print headers/footers
- `--headless=new` flag required for newer Chrome

**Output:** `payments/YYYY-MM/invoice-{no}-{mon}{year}.html` + `.pdf`

### Withholding Tax Form Automation (ภ.ง.ด.3)

Automated system to fill Thai WHT forms for DJ payments.

**Templates saved:** `payments/templates/wht-{djname}.pdf` (11 DJs)
- izaar, krit, ufo, tohmo, manymaur, scotty, benji, jj, linze, april, pound

**Usage:**
```bash
cd payments && source venv/bin/activate
python3 scripts/fill_wht_form.py <dj_name> <amount> <tax> <output_folder> <month> <year> <venue>

# Example:
python3 scripts/fill_wht_form.py pound 9600 480 "2025-12/wht" dec 2025 "CRU & Cocoa XO"
# Creates: pound-dec2025-cru-cocoaxo-wht.pdf
```

**Filename format:** `{dj}-{month}{year}-{venue}-wht.pdf`
- Example: `pound-dec2025-cru-cocoaxo-wht.pdf`
- Makes files easy to search and identify later

**Fields updated automatically:**
- Amount (with comma formatting: 9,600.00)
- Tax withheld (with comma formatting: 480.00)
- Written amount in English (e.g., "FOUR HUNDRED EIGHTY BAHT")
- Date (current date)

**Cross-platform compatibility:**
- Script uses `auto_regenerate=True` to embed font appearances
- Removes `/NeedAppearances` flag to ensure embedded appearances are used
- This fixes comma/decimal rendering issues when opening PDFs on Windows after syncing via Google Cloud

**Payment Timeline:**
- Hotel pays ~1 month after performance period
- December performances → Paid February 1
- January performances → Paid end of February/early March
- February performances → Paid end of March
